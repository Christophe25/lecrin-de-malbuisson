# L'Écrin de Malbuisson – Votre cocon dans le Haut-Doubs

Site promotionnel premium pour un studio de location courte durée situé à Malbuisson (25160), au bord du lac de Saint-Point.

![Aperçu du site](https://raw.githubusercontent.com/Christophe25/lecrin-de-malbuisson/main/images/hero.png)

## ✨ Caractéristiques
- **Design Premium** : Palette inspirée de la nature, typographies soignées et animations au scroll.
- **Calendrier Temps Réel** : Synchronisation iCal avec Airbnb pour afficher les disponibilités.
- **Guide Local** : Sélection de bonnes adresses (restaurants, activités, loisirs) avec photos réelles et adresses exactes.
- **Expérience Mobile** : Optimisé pour iPhone avec icône d'accueil personnalisée (iOS App Icon).
- **Entièrement Responsive** : Adapté aux ordinateurs, tablettes et smartphones.

## 🚀 Installation & Déploiement
Le projet est purement statique (HTML/CSS/JS).

1. Clonez le dépôt : `git clone https://github.com/Christophe25/lecrin-de-malbuisson.git`
2. Ouvrez `index.html` dans votre navigateur.

### Activation de GitHub Pages
Pour rendre le site public :
1. Allez dans l'onglet **Settings** de ce dépôt.
2. Cliquez sur **Pages** (menu latéral).
3. Sous **Branch**, sélectionnez `main` et cliquez sur **Save**.
